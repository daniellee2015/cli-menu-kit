# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.20] - 2026-02-17

### Added
- MINIMAL layout for single-option menus to provide a cleaner, more compact display
- Dynamic hint text that adapts based on actual menu options available

## [0.1.18] - 2026-02-17

### Added
- `preserveOnSelect` configuration option to keep menu rendered after selection instead of clearing it
- Support for preserveOnSelect in boolean, checkbox, and radio menus

### Changed
- Updated .gitignore to exclude .ccb, .ccb-instances, and .claude directories

## [0.1.17] - Previous Release

### Added
- Bold text support for menu items
- Comprehensive configuration documentation in README

## [0.1.16] - Previous Release

### Added
- Configurable UI color system with setUIColors()
